# web chat demo

## How to run

> run server
```angular2html
cd web-chat-demo
npm i
npm start
```

> run client
```
cd web-chat-demo-client
npm i
npm start
```
> open link with [http://localhost:3001](http://localhost:3001/#/?ela=your_first_id)


Should run as desktop app with "npm run build && npm run nw"
