## Elca(Elastos Carrier Addon) API

### Constants

### PresenceStatus:
    PresenceStatus_None = 0
    PresenceStatus_Away = 1
    PresenceStatus_Busy = 2

### ConnectionStatus:
    ConnectionStatus_Connected = 0
    ConnectionStatus_Disconnected = 1

### Data types
    object bootstrap {
        string ipv4,

    object options {
